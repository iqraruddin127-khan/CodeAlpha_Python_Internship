# CodeAlpha Python Programming Internship

## 📌 Overview
This repository contains Python programming projects completed during the CodeAlpha Internship. The projects demonstrate core Python concepts such as file handling, automation, loops, conditionals, and basic chatbot development.

---

## ✅ Completed Tasks

### 🟢 Task 1: Hangman Game
A simple text-based word guessing game where the user guesses letters to find the hidden word.

### 🟢 Task 3: Email Extractor Automation
A Python script that reads a text file and extracts all email addresses using regular expressions and saves them into a new file.

### 🟢 Task 4: Basic Chatbot
A rule-based chatbot that responds to user inputs like "hello", "how are you", and "bye" using conditional statements.
